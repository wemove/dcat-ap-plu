DCAT-AP.PLU Codelisten-Repository / DCAT-AP.PLU Codelist Repository

Dieses Repository enthält die Codelisten, die in der Spezifikation für DCAT-AP.PLU {DCATAPPLU_VERSION} verwendet werden. This repository houses the codelists used in the specification for DCAT-AP.PLU {DCATAPPLU_VERSION} (German).

Die Codelisten können hier abgerufen werden: You can find the available codelists below:

  1. Verfahrensart / Procedure Type
  2. Status eines Verfahrens / Procedure State
  3. Planstatus / Plan Status
  4. Planart / Plan Type
  5. Art eines Verfahrensschrittes / Procedure Step Type
  6. Dokumententyp / Document Type
  7. Datenhaltende Stelle / Authority
Die DCAT-AP.PLU Spezifikation wird auf github gepflegt: https://github.com/wemove/dcat-ap-plu/ The DCAT-AP.PLU specification is being maintained on github: https://github.com/wemove/dcat-ap-plu/.
